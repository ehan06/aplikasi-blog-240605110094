### Deskripsi Aplikasi
Aplikasi ini merupakan pengembangan dari Sistem Manajemen Blog (CMS) Modul 10. Pada proyek ini, arsitektur MVC Laravel diterapkan untuk membangun halaman publik pengunjung (tanpa proteksi login/middleware auth) yang meliputi halaman utama dengan daftar 5 artikel terbaru serta widget kategori, dan halaman detail artikel lengkap beserta widget artikel terkait berdasarkan kategori yang sama.
